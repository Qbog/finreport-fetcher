财报分析台

按 Apple 风格做的简约报表界面：先选公司、时间范围、分类，再一键生成趋势分析、结构分析、同业分析三类图表。图表区域支持方向键浏览。

按当前分析模式使用 ← → ↑ ↓ 切换；点顶部标签切换三大分析类目
