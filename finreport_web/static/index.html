财报分析台

Web 端现在只保留 3 个分析入口：公司类别、时间范围、分析内容。页面会直接读取财报/股价原始数据生成图表，不再依赖旧的图表批处理程序。

趋势：←→ 内容 / ↑↓ 公司；结构：←→ 时间 / ↑↓ 公司；同业：←→ 时间 / ↑↓ 内容

分析设置

分析内容

输出目录：output/_global/web_runs/时间戳/。同一次分析内的趋势/结构/同业/合并图表都会放在全局目录中。

配置辅助

公司类别只在前台创建，保存后会自动写回 config/company_categories.toml，不再需要手工编辑 TOML。

启动时会优先读取 output/_global/company_basics/company_basics.csv 作为公司总表；如果还没有，先运行 python3 -m fincompany_fetcher fetch --out output。
