4) 常见报错与排查 → [TROUBLESHOOTING.md](TROUBLESHOOTING.md)

5) 模板规范与最佳实践 → [TEMPLATE_GUIDE.md](TEMPLATE_GUIDE.md)

8) 测试与 CI → [TESTING.md](TESTING.md)
